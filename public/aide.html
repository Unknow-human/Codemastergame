Comment jouer à CodeMaster ?

But du jeu

Le but est de deviner un code secret de 4 chiffres défini par l'adversaire (ou généré aléatoirement en solo).

Fonctionnement

  • Chaque chiffre doit être compris entre 0 et 9.
  • À chaque essai, le système vous indiquera :
    • Combien de chiffres sont corrects (peu importe la place).
    • Combien de chiffres sont à la bonne position.
  • Exemple :
    Code secret : 3455
    Proposition : 4205
    Résultat : 2 chiffres corrects, 1 bien placé

Modes de jeu

  • Solo : contre l’IA (ou code généré)
  • Duel : contre un autre joueur en ligne

Fonctionnalités supplémentaires

  • Classement global
  • Invitations de duel en temps réel
  • Chat multijoueur
  • Système de scores et abandon
  • Thèmes sombre & clair

Conseils

  • Essayez des combinaisons variées au début
  • Notez les indices pour affiner progressivement
